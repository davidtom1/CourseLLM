# **App Name**: CourseCompanion

## Core Features:

- Student Authentication: Secure login for students with role-based access using Firebase Authentication.
- Teacher Authentication: Secure login for teachers with role-based access to the admin dashboard.
- Socratic Chat: An interface for students to ask questions and receive Socratic guidance from the Course Companion AI, complete with displayed citations from the RAG system.
- Practice Quiz Generation: Students can generate short quizzes on selected topics to test their understanding.
- Topic Mastery Tracking: Display topic mastery levels, helping students identify areas needing more focus.
- Content Upload and Indexing: Teachers can upload course materials (PDF, PPT) for indexing and use in the RAG system.
- Analytics Dashboard: Teachers can view aggregated, anonymized class analytics, including average mastery levels per topic and at-risk student counts.

## Style Guidelines:

- Primary color: Deep sky blue (#3399FF) for a calm and trustworthy educational feel.
- Background color: Very light blue (#E5F2FF) for a clean, uncluttered workspace.
- Accent color: Vivid blue (#3366FF) for important actions and highlights.
- Body and headline font: 'Inter', a sans-serif for a modern and neutral academic interface.
- Use simple, clear icons to represent topics and actions throughout the interface.
- Maintain a clean and responsive layout, optimized for both desktop and mobile devices, focusing on clear information hierarchy.
- Incorporate subtle animations for loading states and transitions to provide feedback without distracting from the content.
